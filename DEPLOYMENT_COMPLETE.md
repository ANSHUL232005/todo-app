## 🆘 COMMON DEPLOYMENT ISSUES

| Problem | Solution |
|---------|----------|
| **Render says "Build failed"** | Check `requirements.txt` in backend; ensure all deps are listed |
| **Frontend can't reach API** | Verify `REACT_APP_API_URL` is set to Render backend URL |
| **500 error on registration** | Check `SECRET_KEY` environment variable on Render |
| **Database empty after redeploy** | SQLite resets on redeploy; use PostgreSQL for persistence |
| **"Port already in use" locally** | Kill: `docker-compose down` then `docker-compose up -d` |
