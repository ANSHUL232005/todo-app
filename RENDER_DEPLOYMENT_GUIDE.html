⚠️ COMMON MISTAKES TO AVOID

❌ DO NOT:
  • Select "Static Site" (wrong type)
  • Select "Web Service (Node)" (use Docker instead)
  • Change port numbers (8000 backend, 3000 frontend)
  • Modify Dockerfile paths
  • Skip environment variables
  • Forget backend URL in frontend REACT_APP_API_URL

🆘 TROUBLESHOOTING

Problem Solution
"Build Failed" Check requirements.txt has all packages; view logs
Frontend blank page Verify REACT_APP_API_URL points to correct backend
Can't register users Check SECRET_KEY environment variable is set
"Cannot reach API" Ensure both services show "Live" status
Database empty after redeploy SQLite resets; use PostgreSQL for persistence
